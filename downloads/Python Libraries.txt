smart door system - Python Libraries :

أول حاجة: تنصيب بايثون

نزّل Python 3.11

أثناء التنصيب تأكد أنك مفاعل خيار 
Add Python to PATH.

-----------------------------------------------------

ثاني حاجة: المكتبات القياسية (ما تحتاجش تنصيب)

الكود يستعمل مكتبات مضمنة في بايثون:

os, time, json, threading, datetime


--------------------------------------------------------

المكتبات الخارجية (لازم تنزلها)

الكود فيه مكتبات بايثون مش تجي معاه، فلازم تنصبهم بالـ pip.

📦 القائمة الكاملة:

opencv-python (باش تخدم cv2)

pip install opencv-python


numpy (للتعامل مع الصور والحسابات)

pip install numpy


pyserial (باش يتعامل مع Arduino عبر الـ Serial)

pip install pyserial


requests (باش يتعامل مع API تليغرام)

pip install requests


face-recognition (للتعرف على الوجه)

ملاحظة: هذي تعتمد على مكتبتين أساسيتين:

dlib

face-recognition-models
عادةً ينزلو مع بعض لما تثبت face-recognition.

التثبيت:

pip install face-recognition


⚠️ تنبيه: dlib مرات يصعب يتبنى على Windows. أسهل طريقة هي تستعمل:

pip install cmake dlib


قبل ما تثبت face-recognition.

Pillow (باش يخدم PIL و ImageTk)

pip install pillow


customtkinter (الـ UI الحديث اللي مستعمل في الكود)

pip install customtkinter

--------------------------------------

4️⃣ مكتبات اختيارية

لو تبي تجرب تشغيل الكاميرا بشكل متقدم ممكن تحتاج:

pip install opencv-contrib-python


مرات تحتاج cmake و wheel قبل تثبيت dlib:

pip install cmake wheel
